# Architecture Decision Records

This section contains public-safe Architecture Decision Records and
summaries. ADRs show how technical direction is decided, constrained,
and reviewed during the modernization program.

## Public ADRs

| ADR | Focus |
|---|---|
| [0001 - LP optimizer architecture](0001-lp-optimizer-architecture.md) | Public tooling architecture and governance boundaries. |

| [0007 - Landing page manager architecture](0007-landing-page-manager-architecture.md) | Public-facing architecture rationale for a transformation artifact. |

## Why This Matters

For transformation leadership, the important signal is not that every
decision is perfect. The signal is that decisions are explicit, reviewed,
traceable, and revisited when evidence changes.

## Not Public

Private ADRs that expose product implementation detail, security
posture, data models, vendor configuration, pricing experiments, or raw
AI execution traces are not mirrored here.
